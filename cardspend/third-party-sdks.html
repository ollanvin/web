SDK 목록

CardSpend는 다음 서드파티 SDK를 사용합니다:

Kakao SDK 2.19.0

용도: 카카오 소셜 로그인

수집 데이터: 이메일, 프로필 (사용자 동의 시)

Google Credential Manager 1.3.0

용도: Google 소셜 로그인

수집 데이터: 이메일, 이름 (사용자 동의 시)

Google Play Billing 7.x

용도: 구독 결제 처리

수집 데이터: 결제 상태 (금융 정보는 Google이 직접 처리)

ZXing (Zebra Crossing)

용도: 바코드/카드 번호 스캔

수집 데이터: 없음 (온디바이스 처리)

CameraX

용도: 카메라 프리뷰 및 캡처

수집 데이터: 없음 (온디바이스 처리)

Google ML Kit (Text Recognition)

용도: 카드 텍스트 인식

수집 데이터: 없음 (온디바이스 처리)

02

데이터 처리 원칙

모든 SDK는 서비스 제공에 필요한 최소한의 데이터만 처리합니다. 광고 또는 사용자 추적 목적의 SDK는 사용하지 않습니다.

SDK List

Data Collected: Email, name (with user consent)

Purpose: Subscription payment processing

Data Collected: Payment status (financial data handled by Google)

ZXing (Zebra Crossing)

Purpose: Barcode/card number scanning

CameraX

Purpose: Camera preview and capture

Data Collected: None (on-device processing)

Google ML Kit (Text Recognition)

Purpose: Card text recognition

Data Collected: None (on-device processing)

02

Data Processing Principles

All SDKs process only the minimum data necessary for service delivery. No SDKs are used for advertising or user tracking purposes.
