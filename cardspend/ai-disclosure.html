01

현재 AI 사용 현황

CardSpend는 현재 인공지능(AI) 또는 머신러닝(ML) 기술을 사용하지 않습니다.

모든 카드 거래 분석은 규칙 기반(rule-based) 알고리즘으로 수행됩니다. 신경망, 대규모 언어 모델, 생성형 AI 등은 사용되지 않습니다.

02

카드 스캔 기능

카드 스캔 기능은 Google ML Kit 또는 CameraX의 텍스트 인식 기능을 사용합니다. 이는 기기 내에서 실행되는 온디바이스 처리이며, 외부 서버로 데이터를 전송하지 않습니다.

03

향후 AI 도입 시 공개 기준

향후 AI/ML 기능을 도입하는 경우, 다음 기준에 따라 공개합니다:

  • 사용되는 AI/ML 기술의 종류 및 목적
  • AI가 처리하는 데이터의 범위
  • 온디바이스 처리 여부
  • 사용자 데이터의 AI 학습 사용 여부 (사용하지 않을 예정)
  • AI 의사결정에 대한 사용자의 거부 권리
04

01

Current AI Usage

CardSpend does not currently use artificial intelligence (AI) or machine learning (ML) technology.

All card transaction analysis is performed using rule-based algorithms. No neural networks, large language models, or generative AI are used.

02

Card Scanning Feature

The card scanning feature uses Google ML Kit or CameraX text recognition. This is on-device processing and does not transmit data to external servers.

03

Future AI Disclosure Standards

If AI/ML features are introduced in the future, we will disclose:

  • Type and purpose of AI/ML technology used
  • Scope of data processed by AI
  • Whether processing is on-device
  • Whether user data is used for AI training (we do not plan to)
  • User's right to opt out of AI-based decisions

当前AI使用状况

CardSpend目前不使用人工智能（AI）或机器学习（ML）技术。

所有银行卡交易分析均使用基于规则的算法。不使用神经网络、大型语言模型或生成式AI。

卡片扫描功能

卡片扫描功能使用Google ML Kit或CameraX的文字识别功能。这是设备端处理，不会将数据传输到外部服务器。

未来AI披露标准

如果未来引入AI/ML功能，我们将披露：

  • 使用的AI/ML技术类型和目的
  • AI处理的数据范围
  • 是否为设备端处理
  • 用户数据是否用于AI训练（我们不打算这样做）
  • 用户拒绝AI决策的权利

01

現在のAI使用状況

CardSpendは現在、人工知能（AI）または機械学習（ML）技術を使用していません。

すべてのカード取引分析はルールベースのアルゴリズムで実行されます。ニューラルネットワーク、大規模言語モデル、生成AIは使用されていません。

02

カードスキャン機能

カードスキャン機能はGoogle ML KitまたはCameraXのテキスト認識機能を使用します。これはオンデバイス処理であり、外部サーバーにデータを送信しません。

03

将来のAI開示基準

将来AI/ML機能を導入する場合、以下の基準に従い開示します：

  • 使用するAI/ML技術の種類と目的
  • AIが処理するデータの範囲
  • オンデバイス処理の有無
  • ユーザーデータのAI学習への使用有無（使用しない予定）
  • AI意思決定に対するユーザーの拒否権
