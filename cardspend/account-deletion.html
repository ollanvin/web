01

삭제 방법

CardSpend 앱 내에서 계정을 삭제할 수 있습니다:

1CardSpend 앱 실행

2하단 탭에서 설정 선택

3멤버십 탈퇴 선택

4확인 버튼을 눌러 삭제 완료

02

삭제 시 처리되는 데이터

계정 삭제 시 다음 데이터가 즉시 영구 삭제됩니다:

계정 정보 (이메일) 카드 정보 (별명, 카드사, 끝 4자리) 거래 기록 백업 데이터 설정값
삭제된 데이터는 복구할 수 없습니다. 삭제 전 필요한 데이터를 백업하십시오.
03

삭제 후 복구

계정 삭제는 즉시 처리되며, 삭제 후 데이터 복구는 기술적으로 불가능합니다.

서비스를 다시 이용하려면 새 계정을 생성해야 합니다.

04

구독 취소

계정 삭제 전 활성 구독을 반드시 취소하십시오. 계정 삭제만으로 구독이 자동 취소되지 않을 수 있습니다.

Google Play: Google Play 앱 → 구독 → CardSpend → 구독 취소

Apple: 설정 → Apple ID → 구독 → CardSpend → 구독 취소

05

문의

계정 삭제 관련 문의: founder@idolab.ai

01

How to Delete Your Account

You can delete your account within the CardSpend app:

1Open the CardSpend app

2Go to Settings from the bottom tab

3Select Delete Membership

4Confirm to complete deletion

02

Data Processed on Deletion

The following data is permanently deleted immediately upon account deletion:

Account info (email) Card info (nickname, issuer, last 4 digits) Transaction records Backup data Settings
Deleted data cannot be recovered. Please back up any needed data before deletion.
03

Recovery After Deletion

Account deletion is processed immediately. Data recovery is technically impossible after deletion.

To use the service again, you must create a new account.

04

Cancel Your Subscription

Please cancel any active subscriptions before deleting your account. Account deletion alone may not automatically cancel subscriptions.

Google Play: Google Play app → Subscriptions → CardSpend → Cancel

Apple: Settings → Apple ID → Subscriptions → CardSpend → Cancel

05

Contact

Account deletion inquiries: founder@idolab.ai

01

删除方法

您可以在CardSpend应用中删除账户：

1打开CardSpend应用

2在底部标签中选择设置

3选择注销会员

4确认完成删除

02

删除时处理的数据

账户删除时以下数据将立即永久删除：

账户信息（邮箱） 卡片信息（昵称、发卡机构、末4位） 交易记录 备份数据 设置信息
已删除的数据无法恢复。请在删除前备份所需数据。
03

删除后恢复

账户删除即时处理，删除后数据在技术上无法恢复。

如需再次使用服务，须创建新账户。

04

取消订阅

删除账户前请务必取消活跃订阅。仅删除账户可能不会自动取消订阅。

Google Play：Google Play应用 → 订阅 → CardSpend → 取消订阅

Apple：设置 → Apple ID → 订阅 → CardSpend → 取消订阅

05

01

削除方法

CardSpendアプリ内でアカウントを削除できます：

1CardSpendアプリを開く

2下部タブから設定を選択

3メンバーシップ退会を選択

4確認ボタンを押して削除完了

02

削除時に処理されるデータ

アカウント削除時、以下のデータが即座に永久削除されます：

アカウント情報（メール） カード情報（ニックネーム、カード会社、下4桁） 取引記録 バックアップデータ 設定値
削除されたデータは復旧できません。削除前に必要なデータをバックアップしてください。
03

削除後の復旧

アカウント削除は即時処理され、削除後のデータ復旧は技術的に不可能です。

サービスを再度利用するには、新しいアカウントを作成する必要があります。

04

サブスクリプションの取消

アカウント削除前に有効なサブスクリプションを必ず取り消してください。アカウント削除だけではサブスクリプションが自動取消されない場合があります。

Google Play：Google Playアプリ → 定期購入 → CardSpend → 定期購入を解約

Apple：設定 → Apple ID → サブスクリプション → CardSpend → サブスクリプションをキャンセル

05

お問い合わせ
